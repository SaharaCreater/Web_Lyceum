Проект по WebServer + API "Telegram-бот с географическими тестами"

Авторы: Дунаев Александр, Лопатин Степан, Шевцов Захар

Идея проекта: Сделать бота в telegram, который проверяет знания географии у пользователя.

Реализация:
Проект сделан на языке Python. В нём использованы библиотеки json, logging, sqlite3 и random.
Реализованы разные уровни сложности для географических тестов, ведение статистики пользователей, её хранение в базе данных.
Спутниковые снимки хранятся в папке data. Они используются в тестах с помощью .json файла.
Бот имеет  несколько команд:
При команде /start бот напишет вам приветствующее сообщение и предоставит вам клавиатуру с разными командами.
Команда /test начинает тест
Команда /level позволяет выбрать уровень сложности теста
Команда /stats показывает статистику пользователя
Команда /help напоминает пользователю о том, какие функции содержит бот

Итог:
Проект представляет собой telegram-бота с географическими тестами, проверяющего знания географии у пользователя.
Показываются лучшие результаты среди лёгких, средних и сложных тестов.
Данные о лучших результатах всех пользователей сохраняются в базе данных.
